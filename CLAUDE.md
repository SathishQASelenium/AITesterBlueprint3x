# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview
This repository serves as a learning and documentation hub for "The Testing Academy - AI Tester Blueprint 3x". It is not a software project but a collection of study materials, notes, and task deliverables.

## Repository Structure
- `chapter_XX_[Topic]/`: Directories organized by course chapters (e.g., `chapter_01_LLM_Basics`).
- `chapter_XX_[Topic]/Tasks/`: Contains subdirectories for specific course assignments, including PRDs, research reports, and screenshots.
- `Notes.md`: High-level tracking of topics learned and task progress.

## Development & Maintenance
Since this is a documentation repository, there are no build, lint, or test commands. 

### Common Operations
- **Adding Materials**: Place new learning resources (HTML, PDFs, etc.) in the relevant chapter directory.
- **Submitting Tasks**: Create a new folder under the chapter's `Tasks/` directory for each assignment and place all relevant deliverables inside.
- **Updating Progress**: Keep `Notes.md` updated with a summary of learned topics and completed tasks.
